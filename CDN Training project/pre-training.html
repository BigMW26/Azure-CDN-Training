Install Visual Studio Code https://code.visualstudio.com/

Configure Extentions

Azure Tools

Azure Account (Should automatically install with Azure Tools)

Sign into Azure from extension

Deploying the test App

Download app from GitHub: https://github.com/BigMW26/Azure-CDN-Training

Extract with parent folder

Load parent folder into Visual Studio Code

Deploy application to subscription

Name it <alias>CDN

Visit the site with the your <alias>cnd.azurewebsites.net

Deploy CDN and create WebApp Endpoint

Follow instructions on Frontend documentation https://docs.microsoft.com/en-us/azure/cdn/cdn-add-to-web-app#create-a-cdn-profile-and-endpoint

Install the following Tools:

  • Fiddler (v4.4.9.2 or higher) - https://www.telerik.com/fiddler

  • FDTraces - \\tkfiltoolbox\tools\FDTraceInstaller\1.0.10

    • Run file as administrator from location

  • Install WGET or CURL

    • WGET - http://gnuwin32.sourceforge.net/packages/wget.htm

      Install WGET by downloading the complete setup package

  • CURL http://curl.haxx.se/

  • Kusto
